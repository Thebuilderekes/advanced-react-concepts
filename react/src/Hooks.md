## Rules of react hooks

- You can only call hooks from a react component, not regular javascript function

- You can only call hooks at the top level before the return statement, not within a conditional statement.

- You can call mulitple state or effect hooks and there must always be in the sequence

- You can make multiple calls in the same sequence
